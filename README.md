# 白日梦计划 V2.2 archive book index board

本版把当前结构修正为“蓝色白日梦档案本目录系统”：

- 封面：蓝色白日梦档案本，被透明塑封膜封住。
- 目录：同一 100svh 画布内，上半为当前档案预览，下半为半页 INDEX 目录抽屉。
- 目录交互：点击条目先切换上半屏预览，再通过预览按钮进入对应页面。
- 福里：先进入一屏档案总览，再继续翻看三次显影。
- 塑封：保留照片、本地文字、随机档案章、判词和回执编号。
- 投递口：保留问题卡和低门槛回复。

文件结构保持：index.html / styles.css / app.js / assets / README_改动说明.md / SELF_CHECK_v2.2.txt。
